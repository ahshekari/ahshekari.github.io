Project Overview

This project was part of the Cloud Business Development course at BCIT. Our team, Infinite Dynamics (Amir Shekari, Pouria Pazhouhesh, Ghazal Mirzad), researched, designed, and deployed a live web application using Microsoft Azure App Service. We compared deployment strategies across different platforms while focusing on business scalability and cost-effectiveness.

What We Did

  • Developed a web application using HTML, CSS, and JavaScript.
  • Deployed the app to Azure App Service with CI/CD pipelines from GitHub.
  • Explored alternative deployment options like Vercel and GitHub Pages for redundancy.
  • Analyzed hosting trade-offs: self-hosting vs cloud-based vs managed hosting.
  • Presented business use cases for leveraging cloud technologies in small-to-medium businesses.

Impact

  • Delivered a live running application demonstrating Azure‚Äôs scalability.
  • Showcased team collaboration via GitHub version control and branching workflows.
